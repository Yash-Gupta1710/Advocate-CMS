# ⚖️ Lexora

### Modern Legal Practice Management Platform

Lexora is a full-stack legal practice management system designed to help advocates and law firms streamline client management, appointment scheduling, case tracking, document handling, and legal workflows through a secure and intuitive platform.

---

## 🚀 Features

### 🔐 Authentication & Security

- JWT-based Authentication
- Secure Login & Registration
- Role-Based Access Control
- Password Recovery using Security Questions
- Protected API Endpoints

### 👨‍⚖️ Lawyer Portal

- Dashboard Analytics
- Client Management
- Case Tracking
- Appointment Management
- Document Management
- Hearing Tracking
- Notification Center

### 👤 Client Portal

- Appointment Booking
- Case Progress Tracking
- Secure Document Access
- Profile Management
- Real-Time Status Updates

### 📂 Case Management

- Create & Manage Cases
- Priority Classification
- Status Tracking
- Hearing Timeline Management
- Case History

### 📄 Document Management

- Secure File Upload
- Document Storage
- Download & Retrieval
- Client-Lawyer File Sharing

### 📊 Analytics Dashboard

- Active Cases
- Pending Requests
- Appointment Statistics
- Client Overview
- Practice Insights

## ⚙️ Local Setup

### Clone Repository

```bash
git clone https://github.com/Yash-Gupta1710/Advocate-CMS.git
cd Advocate-CMS
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

Backend runs at:

```text
http://localhost:8080
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs at:

```text
http://localhost:3000
```
